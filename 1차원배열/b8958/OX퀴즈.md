```JavaScript
for(let j =0; j<testcase.length; j++) {

}
```
이렇게 주는게 아니라 각 testcase에 길이만큼 반복을 해야 해당 테스트 케이스 내에 문자를 비교를 해주니까 testcase[i].length 이렇게 해주어야 잘 반복을 할수 있다 (내가 스스로 생각한거임 답 안봄 ㅇㅇ ㅋㅋㅋㅋ)
